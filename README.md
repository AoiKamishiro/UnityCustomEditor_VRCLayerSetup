<H3>起動方法</H3>

<H3>プログラムについて</H3>
<a>VRChat用のレイヤー設定をセットアプ出来ない、Avatar3.0用SDKでレイヤーをセットアップできます。</a>
<H4>レイヤー設定をするメリット</H4>
<a>カメラやライトなど、レイヤーに依存しているコンポーネントをアバターに仕込む時に、既存のプロジェクトなどからレイヤー設定を参照したりする必要がなくなります。</a>
